HTML - table#

table의 기본 구조

  • <table></table> : 표 정의 태그
  • <tr></tr> (table row) : 표의 행(가로줄)
  • <th></th>(table head) : 표의 헤드라인. 디폴트로 굵은 글씨
  • <td></td> : <tr></tr> 태그 내부에서 사용된다. 행의 열의 값을 채워 넣는다.
<h1>표(table) 형식으로 정보를 출력하기</h1>
<table>
    <tr>
        <th>번호</th>
        <th>이름</th>
        <th>주소</th>
    </tr>
    <tr>
        <td>1</td>
        <td>김이박</td>
        <td>김해</td>
    </tr>
    <tr>
        <td>2</td>
        <td>사람</td>
        <td>부산</td>
    </tr>
</table>

table의 여러 기능

  1. border : 표의 테두리 그리기 (선이 2개로 나옴)
<table border="1"></table>
  1. caption : 표의 제목 
<table>
    <caption>회원 목록</caption>
</table>

  1. colgroup :
<table>
    <!--칼럼의 폭을 조절하고싶으면, colgroup 요소를 활용-->
    <colgroup>
        <col width="100">
        <col width="200">
        <col width="300">
    </colgroup>
</table>
  1. 표의 영역 나누기 (Step08_table2.html)
  • thead : 표의 헤드라인 부분 설정
  • tbody : 표의 내용.
  • tfoot : 합계 등의 통계에 관련된 내용 표기.
    -> tfoot은 일반적으로 thead 바로 아래에 둔다
    -> tfoot은 어느 위치에 들어가도 실제 표에서 맨 아래에 위치함
<table>
    <thead>
        <tr>
            <th>번호</th>
            <th>이름</th>
            <th>주소</th>
        </tr>
    </thead>
    <tfoot>
        <tr>
            <th>합계</th>
            <td>4,300원</td>
            <td>800원</td>
        </tr>
    </tfoot>
    <tbody>
        <tr>
            <td>1</td>
            <td>김이박</td>
            <td>김해</td>
        </tr>
        <tr>
            <td>2</td>
            <td>사람</td>
            <td>부산</td>
        </tr>
    </tbody>
</table>
  1. 표의 컬럼(셀) 합치기 (Step08_table3.html)
  • colspan : 열 합치기(가로로 합치기)
  • rowspan : 행 합치기(세로로 합치기)
  • 행, 열을 한꺼번에 합칠 수 있음
  • 사용법
<td colspan="합칠 컬럼 개수"></td>
<td rowspan="합칠 컬럼 개수"></td>
<!-- 행, 열 둘 다 합치기 -->
<td colspan="합칠 컬럼 개수" rowspan="합칠 컬럼 개수"></td>
  • ex)
<table border="1">
    <tr>
        <td colspan="2">1</td>  <!-- colspan="합칠 컬럼의 개수"-->
        <!-- ctrl + / : 자동으로 주석 -->
        <!-- 아래 <td>2</td>는 위의 1이 합쳐지면서 불필요하여 주석처리 함 -->
        <!-- <td>2</td> -->
        <td>3</td>
        <td>4</td>
        <td>5</td>
    </tr>
</table>

-> colspan=“합칠 컬럼의 개수” 

<table border="1">
    <tr>
        <td rowspan="2">1</td>
        <td>2</td>
        <td>3</td>
        <td>4</td>
        <td>5</td>
    </tr>
    <tr>
        <!-- <td>6</td> -->
        <td>7</td>
        <td>8</td>
        <td>9</td>
        <td>10</td>
    </tr>
</table>

-> colspan=“합칠 컬럼의 개수”

<h1>1,2,6,7 합치기</h1>
<table border="1">
    <tr>
        <td colspan="2" rowspan="2">1</td>
        <!-- <td>2</td> -->
        <td>3</td>
        <td>4</td>
        <td>5</td>
    </tr>
    <tr>
        <!-- <td>6</td> -->
        <!-- <td>7</td> -->
        <td>8</td>
        <td>9</td>
        <td>10</td>
    </tr>
</table>

-> 행, 열을 한꺼번에 합칠 수 있음


table의 style 지정(head 요소에 style 지정 방식)

<style>
    /*여기는 css 영역*/
    /* 특정 id를 가진 table에 style 지정 */
    #one{ /* id 속성의 value 가 one 인 요소를 선택 */
        border-collapse: collapse;
    }
    /* 모든 table 전체에 style 지정 */
    table{
        border-collapse: collapse;
        background-color: yellow;
    }
</style>



☆ border-collapse
: table에서 border을 사용한 테두리는 두 줄로 표현되는데, 이를 하나의 선으로 표시하기 위해서 사용하는 style

<style>
    table{
        border-collapse: collapse;
    }
</style>

<table id="one" border="1"></table>




HTML - form#

form

  • client가 입력하여 서버로 전송하는 양식
  • action=“서버에 전송할 위치(요청 경로)”
  • action이 없을 수도 있음
<form action="">
    <input>
    <label></label>
    <button></button>
    ...
</form>

input 참고-MDN_Web_Docs

<input type="" id="" name="">
  • type : input type 지정
  1. text
  2. password : 실제 입력되는 내용이 보이지 않음
  3. button
  4. radio : 같은 name 그룹끼리 하나만 선택 가능
    • checked 속성 : 기본적으로 하나가 체크가 되어 있게 할 수 있음
  • name : input에서 여러개의 정보가 전송 될 때, 서버에서 구분을 위해 쓰임 
    //서버에서 받는 예시
request.getParameter("eamil");
request.getParameter("pwd");

label(레이블)

  • input에 어떠한 내용을 넣는지 이름을 지정.
  • 시각에 불편함이 있을 경우, 포커스가 이동하면 label의 이름을 읽어줌
  • label의 for=“id” : label의 for의 속성값과 input의 id 속성의 값이 동일하게 작성되어야 한다.
<label for="email">이메일</label>
<input type="text" id="email" name="email">
  • label에서 for속성을 제거하고, input 부분을 포함시키므로써 뒤의 innerText를 label의 text로 지정할 수 있다.
    ex) 뒤의 ‘남자’를 읽어줌
<label>
    <input type="radio" name="gender" value="man" checked> 남자
</label>

button

<button type="submit">로그인</button>
  • type
  1. submit : 서버에 데이터 전송 버튼
  2. reset : 입력한 데이터 초기화

fieldset - legend

  • 다른 요소들을 필드로 묶어서, legend의 innerText의 그룹으로 묶는다
  • 예시)
성별 정보 선택 
<!-- Step09_form2.html -->
<fieldset>
    <legend>성별 정보 선택</legend>
    <label>
        <input type="radio" name="gender" value="man" checked> 남자
    </label>
    <label>
        <input type="radio" name="gender" value="woman"> 여자
    </label>
    <label>
        <input type="radio" name="gender" value="etc"> 기타
    </label>
</fieldset>

radio

  • 같은 name 그룹끼리 하나만 선택 가능
  • checked 속성 : 기본적으로 하나가 체크가 되어 있게 할 수 있음
  • value에는 한글 사용 가능

남자 여자 

<input type="radio" name="gender" value="man" checked> 남자
<input type="radio" name="gender" value="woman"> 여자

checkbox

  • 중복 선택 가능하다.
  • name으로 한 그룹을 묶을 수 있음
  • value에는 한글 사용 가능



축구 베이킹 피아노 기타연주

<input type="checkbox" name="bobby" value="축구"> 축구
<input type="checkbox" name="bobby" value="베이킹"> 베이킹
<input type="checkbox" name="bobby" value="피아노"> 피아노
<input type="checkbox" name="bobby" value="기타연주"> 기타연주

select : 드롭다운(drop down) 모양

  • 선택하면 아래로 목록이 열리는 드롭다운 모양
  • option에 value가 없으면 innerText가 전송


<select name="job" id="job">
    <option value="">선택</option>
    <option value="programmer">프로그래머</option>
    <option value="doctor">의사</option>
    <option value="piaist">피아니스트</option>
    <option value="etc">기타</option>
</select>

file : 첨부파일

  • file 선택하여 첨부할 수 있다.


<input type="file" id="myFile" name="myFile">

textarea : 메모장

  • 속성의 cols="" rows="" 는 크기를 지정
  • css로도 지정 할 수 있음(style 부분에 지정)



<textarea name="comment" id="comment" ></textarea>
<!-- 위의 회색 추가 -->
<textarea name="comment" id="comment" cols="30" rows="10" style="background-color: #e2e2e2;"></textarea>

color(색상)

  • 웹 브라우저에 따라서 색상 선택 방법이 다르게 나옴(인터넷 익스플로러는 사용 X)
  • 서버로 넘어가는 값은 색의 값


<!-- Step09_form3.html -->
<input type="color" id="color" name="color">

range(범위)

  • 웹 브라우저에 따라서 선택 방법이 다르게 나옴
  • 실제로 서버로 너어가는 값은 바의 움직임이 멈춘 곳의 숫자가 들어간다.
  • 속성
    • min : 최솟값
    • max : 최댓값
    • step : 눈금 1개의 값
    • value : 기본값



<input type="range" id="range" name="range" min="10" max="100" step="1" value="50">

date(날짜)

  • 직접 입력 가능 & 달력 선택 가능 : 크롬 (인터넷 익스플로러 안됨)


<input type="date" id="date" name="date">

time(시간)

  • 직접 입력 가능 & 숫자 화면 선택 가능 : 크롬(나머지는 모르겠음)


<input type="time" id="time" name="time">

number(숫자)

  • 숫자 외의 입력은 불가능하다
  • 크롬 : 맨 오른쪽에 1씩 올리고, 내릴 수 있는 화살표로 조절 가능.
  • 속성
    • min : 최솟값
    • max : 최댓값
    • step : 한 칸 당 올라가는 숫자. 유효한 값은 min + step * n 이다.


<input type="number" id="number" name="number" min="0" max="100" step="2">




HTML - a#

a(anchor)

  • 하이퍼링크, 책갈피, javascript 등을 수행할 때 사용
  • #id 를 통해서 링크로 이동할 수 있음
<a href="#id">one</a>
  • 기본 형태
<a href="주소"> innerText-클릭하면 주소로 이동</a>
  • ex)
<a href="hello.html">hello.html로 이동</a>
<a href="https://daum.net">daum으로 이동</a>
<a href="https://naver.com">naver로 이동</a>
  • 그림에 링크를 넣을 수 있음
<a href="hello.html">
    <img src="images/1.jpg" alt="" width="300px">
</a>
  • inline(인라인)요소는 원래 블럭 요소를 자식요소로 가질 수 없다.
    단, a 요소만 예외적으로 div 같은 블럭 요소로를 자식 요소로 가질 수 있다.
<!-- Step10_anchor.html -->
<!-- 아래의 예제에서는 block 전체의 어디를 누르던 링크를 타고 갈 수 있도록 만들기 위해 사용했다. -->
<a href="hello.html">
    <div>
        <h3>어쩌구 어쩌구</h3>
        <p>
            <img src="images/01_iron_man_2.jpg" alt="" width="300px">
            <!-- lorem : 히브리어? 아무런 글자가 나온다. 그냥 빈칸 채우기 용 -->
            Lorem ipsum dolor sit amet consectetur adipisicing elit. Nemo labore beatae voluptate porro maxime ullam unde debitis nisi at eaque quod delectus nam est, voluptatibus quia dolore ipsa hic! Blanditiis!
        </p>

    </div>
</a>

  • 또 다른 기능(추가) - Step10_anchor3.html
  1. javascript 링크를 누를 때마다 실행
<a href="javascript:alert('고만 좀 눌러라')">눌러보셈</a>

  1. 전화 걸기
<a href="tel:010-1111-2222">전화 걸기</a>

  1. 이메일 쓰기
<a href="mailto:aaa@naver.com">이메일 쓰기</a>




기타#

class

  • 요소의 속성 중 하나
  • 하나의 그룹을 형성 : id와는 다르게 class는 여러 요소에 하나의 class명을 사용하여 그룹을 만들 수 있다.
  • class명으로 style을 지정하고 싶을 때 -> ‘.클래스명’을 사용한다.
<head>
    <style>
        .container{
            width: 768px;
            /* 좌, 우의 margin을 auto로 바꾸면 -> 가운데 정렬 가능 */
            /* 아래 두 줄은 -> margin:0 auto; 로 바꿔 쓸 수 있다. */
            margin-left: auto;
            margin-right: auto;
            background-color: yellow;
        }
    </style>
</head>



div 추가

  • div에 width를 지정하여 범위를 줄여도 block 자체의 범위는 줄어들지 않아, 빈 칸에도 하이퍼링크가 적용된다.

li 여러개 생성하는 명령어

<ul>

  • ul>li*n(li 개수)
    : ul의 자식요소 li를 n개 자동으로 생성
ul>li*5
<ul>
    <li></li>

    <li></li>
    <li></li>

</ul>

    : ul의 자식요소 li의 innerText를 자동으로 숫자를 1부터 순서대로 생성
ul>li{$}*5
<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>

p#two

p#three{three}
