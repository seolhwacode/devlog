웹 브라우저에서 개발자 도구 - 검사 도구#

검사 도구 : f12(ctr+shift+i)

  • 화면을 나타내는 html 파일을 표시.
  • 맨 왼쪽 위의 선택을 활용하여 아래의 styles에서 테스트 할 수 있다.
  • console 창 : javascript를 테스트 수행 할 수 있다. 오류 또한 출력한다.



html 기초#


html 기본 요소들

  1. html5 형식의 문자열이라고 웹브라우저에 알리기
<!DOCTYPE html>
  1. 주요 언어 설정 : en(영어), ko(한글)
<html lang="en">
  1. head요소 : head에는 여러가지 설정 정보가 들어간다.

  • style : css(desing)영역은 style요소를 이용해서 만듬

  • script : javascript(action)영역은 script요소를 이용해서 만듬. script는 body 내부에도 들어갈 수 있다.

      ☆ head와 body에 javascript 위치에 따른 차이점
  -> 실행 순서는 head 부터 아래로 이루어짐. 고로, head의 script가 진행 후에 body의 스크립트가 진행됨

  • <title></title> : 웹 페이지의 제목(탭 부분 표기)

  • <meta charset=""> : 인코딩 방식

  • innerText : 태그 사이에 들어가는 text. 실제로 웹 페이지에 text로 표기됨.

<head>
   <meta charset="UTF-8">
   <meta http-equiv="X-UA-Compatible" content="IE=edge">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <!-- 이 웹페이지의 제목(탭의 제목)-->
   <title>hello 페이지</title> <!-- innerText -->
</head>
  1. body요소 : body 에는 웹브라우저에 표시할 내용을 주로 나열한다.
<!-- body 에는 웹브라우저에 표시할 내용을 주로 나열한다.-->
<body>
    <h1>안녕하세요 이제 시작이군요.</h1>
    <h2>중제목 입니다</h2>
    <h3>소제목 입니다.</h3>
    <p>단락 입니다</p>
    <div>문단 입니다</div>
    <i>이텔릭체 입니다</i>
</body>
</html>

태그의 기초
1) element name : 요소명 ex) <h1></h1>
2) attribue : 속성 ex) <html lang=“en”>

  • attribute name(속성명) ex) lang
  • attribute value(속성값) ex) “en”

부모 요소 / 자식 요소 : 요소는 요소로 감쌀 수 있다.

  • 부모 요소 : 겉에서 다른 요소를 감싸는 요소
  • 자식 요소 : 부모 요소로 감싸진 요소

단일 요소

: </> 닫는 태그가 필요 없는 요소. 단독으로 존재
ex) <meta charset=“UTF-8”>

style 요소

  • css 영역은 style요소를 이용해서 만듬
  • css에서 주석 : /* */
    ex)
<head>
    <style>
        /* 여기는 css (stylesheet) 영역 입니다. */
        h1{
            color:blue;
            background-color: yellow;
        }
        p{
            color:red;
        }
    </style>
</head>

script 요소

  • 기본적으로 javascript 코드가 들어간다.
  • head 요소와 body 요소에 들어갈 수 있다.
  • 주석 : //(한 줄 주석), /**/(여러줄 주석)
<script lang="ts">
    //여기는 javascript입니다.
    /* javascript 영역에서 여러줄 주석*/
    alert("안녕하세요");
</script>



html body 요소#

: 화면에 표현하는 다양한 요소들이 존재한다.


요소들의 공통적인 속성

  1. class : 그룹
    - 그룹으로 style 지정, array로 만들기 등이 가능하다.
    - ‘.’ : style, querySelector 등에 사용할 때 맨 앞에 사용한다.
    ex) .className

    .two {
    width: 500px;
}

  2. id : 아이디
    - 아이디는 중복될 수 없다.
    - 띄어쓰기를 통해서 아이디를 여러개 지정할 수 있다.
    - ‘#’ : style, querySelector 등에 사용할 때 맨 앞에 사용한다.
    ex) #id

    #one{
    width: 200px;    
}
    • style의 적용 우선순위 : id(아이디가 여러개 지정되어 있다면, 가장 마지막 id의 style이 적용됨) > class > element(요소)s

  3. display

  • blcok : blcok element(요소). 위에서 아래로 하나식 쌓인다. 폭은 화면 전체
    • ex) div, h1~h6, p(단락) 등
  • inline : inline element(요소). 왼쪽에서 오른쪽으로 하나씩 쌓인다.
    • 인라인 요소는 글에 영향이 없이 일부에 다른 속성을 부여하기 위해 사용한다.
    • ex) span(각종 스타일 넣을 수 있음), i(이텔릭체), em, b(굵은 선), string(강조 + 굵은선)
  1. 요소 여백
  • [ top | right | bottom | left ]
    • padding : 안쪽 여백
    • border : 안과 밖의 경계
    • margin : 바깥 여백

제목 : h

  • 숫자가 커질 수록 글자가 작아진다

  • 1~6까지 존재한다.

    • <h1></h1> : 대제목
    • <h2></h2> : 중제목
    • <h3></h3> : 소제목

  • 위아래 margin은 겹쳐진다.

  • block 요소


단락 : p

<p> innerText </p>
  • 문자열 단락(paragraph)을 구성할 때 사용
  • block 요소

문단 : div

<div> innerText </div>
  • 문단을 나타낼 때 사용
  • 주로 다른 요소를 포함하는 용도로 사용
  • block 요소
  • 일반적으로 block 요소는 다른 block요소를 자식 요소로 둘 수 없지만, div 요소는 다른 block 요소를 자식요소로 가질 수 있다.
  • div에 width를 지정하여 범위를 줄여도 block 자체의 범위는 줄어들지 않아, 빈 칸에도 하이퍼링크가 적용된다.

개행 : br

<br>

  • 개행 요소. 단순히 한 줄 개행한다.

    ☆ div와 br 차이

    • br은 단순 개행
    • div는 브라우저의 가로 전체를 포함하며, div로 둘러싸인 내용을 블록으로 감싼다

이미지 : img

<img src="" alt="">
  • src : 로딩할 이미지의 경로
  • alt : 이미지에 대한 자세한 설명
  • inline 요소
  • 단일 요소(닫는 태그가 필요 없음)
  • 이미지가 경로에 존재하지 않거나, 경로가 다르다면 사진은 빈칸으로 alt의 설명이 표기됨
  • 시각이 불편한 사람들을 위해 alt의 설명을 읽어줌

span

<span> innerText </span>
  • 대표적인 inline 요소
  • 글에 영향이 없이 일부에 다른 속성을 부여하기 위해 사용
  • ex)
<p>하나 둘 셋 span 요소는 <span id="four">인라인</span> 요소 입니다.</p>

굵은 글씨 : b

<b> innerText </b>
  • b 요소는 단순히 굵은 글씨

강조 : strong

<strong> innerText </strong>
  • 굵은 글씨 + 강조의 의미를 가지고 있음

이텔릭체 : i

<i> innerText </i>
  • 단순한 이텔릭체

이텔릭 + 강조 : em

<em> innerText </em>
  • 이텔릭체 + 강조의 의미를 가지고 있음

순서 없는 목록(unordered list) : ul

<ul>
    <li> a </li>
    <li> b </li>
    <li> c </li>
</ul>
  • display : block 블록 요소이다.
  • list-style-type : 기본은 disc(앞에 점 표시)
  • margin-top, margin-bottom, padding-left(왼쪽 패딩)이 기본적으로 존재
  • 맨 앞에 disc가 붙는다.(점)

순서 있는 목록(ordered list) : ol

<ol>
    <li> a </li>
    <li> b </li>
    <li> c </li>
</ol>
  • 목록 앞에 disc대신 숫자가 붙는 것을 제외하고는 ul 요소와 유사
  • disc 대신에 숫자가 붙는다

정의형 목록 (Definition List) : dl

<dl>
    <dt>title1</dt>
    <dd>explain1</dd>

    <dt>title2</dt>
    <dd>explain2</dd>
</dl>
  • dt : 제목
  • dd : 제목에 대한 설명(디폴트로 들여쓰기)
  • ex)
<dl>
    <dt>HTML</dt>
    <dd>Hyper Text Markup Language</dd>

    <dt>CSS</dt>
    <dd>Desing 적인 요소를 결정합니다.</dd>

    <dt>javascript</dt>
    <dd>language 적인 요소를 담당합니다.</dd>
</dl>
